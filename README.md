# Seminar / Workshop Presentation Slides

Repositori ini berisi kumpulan **slide presentasi (PPT)** yang saya gunakan dalam kegiatan **seminar/workshop** yang telah dilaksanakan sebelumnya. Materi disusun untuk tujuan edukasi, sharing pengetahuan, dan diskusi teknis sesuai dengan topik acara.

Materi disesuaikan dengan audiens pada saat acara berlangsung dan **tidak selalu mencerminkan kondisi terbaru**.
